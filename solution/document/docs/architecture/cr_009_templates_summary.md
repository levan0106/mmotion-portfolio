# CR-009: Financial Freedom Planning Templates - Summary

## Tổng quan

Hệ thống cung cấp **7 templates** phổ biến để người dùng dễ dàng bắt đầu lập kế hoạch tài chính. Mỗi template đã được cấu hình sẵn với các giá trị mẫu phù hợp và chỉ rõ biến nào sẽ được tính toán.

---

## Template 1: Tiết kiệm và đầu tư (Savings & Investment)

### Mô tả
Dành cho người đang làm việc, muốn tích lũy và đầu tư để đạt mục tiêu tài chính trong tương lai.

### Thông số mẫu

| Thông số | Giá trị | Ghi chú |
|----------|---------|---------|
| **Mục tiêu** | Tính từ chi phí sinh hoạt | |
| - Chi phí/tháng | 50,000,000 VND | Có thể điều chỉnh |
| - Tỷ lệ rút | 4% (4% rule) | |
| - → Tự động tính | **15,000,000,000 VND** | (50M × 12) / 0.04 |
| **Vốn ban đầu** | 500,000,000 VND | |
| **Tiền bổ sung** | 10,000,000 VND/tháng | |
| **Tần suất** | Hàng tháng | |
| **Loại** | Bổ sung (contribution) | |
| **Thời gian đầu tư** | 15 năm | |
| **Tỉ suất lợi nhuận** | **[Để trống - sẽ tính]** | ⚠️ Biến cần tính |
| **Lạm phát** | 3.5%/năm | Mặc định VN |
| **Mức rủi ro** | Moderate (Cân bằng) | |

### Biến sẽ tính
**Required Return Rate (RRR)** - Tỉ suất lợi nhuận yêu cầu

### Hướng dẫn sử dụng
- Template này phù hợp cho người đang làm việc, có thu nhập ổn định
- Điều chỉnh chi phí sinh hoạt theo tình hình thực tế của bạn
- Tăng tiền bổ sung hàng tháng sẽ giảm tỉ suất lợi nhuận yêu cầu
- Có thể kéo dài thời gian đầu tư để giảm áp lực về lợi nhuận

### Ví dụ kết quả
- **RRR tính được:** ~12.5%/năm
- **Future Value:** ~8.37 tỷ VND (sau 15 năm với lạm phát 3.5%)

## Template 2: Đã nghỉ hưu, rút tiền (Retirement Withdrawal)

### Mô tả
Dành cho người đã nghỉ hưu, có khoản tiết kiệm và muốn rút tiền định kỳ để chi tiêu.

### Thông số mẫu

| Thông số | Giá trị | Ghi chú |
|----------|---------|---------|
| **Mục tiêu** | Tính từ chi phí sinh hoạt | |
| - Chi phí/tháng | 30,000,000 VND | |
| - Tỷ lệ rút | 4% (4% rule) | |
| - → Tự động tính | **9,000,000,000 VND** | (30M × 12) / 0.04 |
| **Vốn ban đầu** | 10,000,000,000 VND | Đã có sẵn |
| **Tiền rút** | -30,000,000 VND/tháng | ⚠️ Số âm (rút tiền) |
| **Tần suất** | Hàng tháng | |
| **Loại** | Rút tiền (withdrawal) | |
| **Thời gian đầu tư** | 25 năm | |
| **Tỉ suất lợi nhuận** | **[Để trống - sẽ tính]** | ⚠️ Biến cần tính |
| **Lạm phát** | 3.5%/năm | |
| **Mức rủi ro** | Conservative (An toàn) | |

### Biến sẽ tính
**Required Return Rate (RRR)** - Tỉ suất lợi nhuận tối thiểu để portfolio không cạn kiệt

### Hướng dẫn sử dụng
- Template này phù hợp cho người đã nghỉ hưu, cần rút tiền hàng tháng
- Tỷ lệ rút 4% là an toàn cho hầu hết trường hợp
- Nên đầu tư an toàn để bảo toàn vốn
- Theo dõi portfolio thường xuyên để đảm bảo không cạn kiệt

### Ví dụ kết quả
- **RRR tính được:** ~4-5%/năm (để duy trì portfolio)
- **Mục tiêu:** Đảm bảo portfolio không cạn kiệt trong 25 năm

## Template 3: Tự do tài chính sớm (Early Retirement / FIRE)

### Mô tả
Dành cho người muốn nghỉ hưu sớm (trước 50 tuổi), cần tích lũy nhanh để đạt tự do tài chính.

### Thông số mẫu

| Thông số | Giá trị | Ghi chú |
|----------|---------|---------|
| **Mục tiêu** | Tính từ chi phí sinh hoạt | |
| - Chi phí/tháng | 40,000,000 VND | |
| - Tỷ lệ rút | 3.5% | An toàn hơn cho thời gian dài |
| - → Tự động tính | **13,714,285,714 VND** | (40M × 12) / 0.035 |
| **Vốn ban đầu** | 1,000,000,000 VND | |
| **Tiền bổ sung** | 20,000,000 VND/tháng | Đầu tư mạnh |
| **Tần suất** | Hàng tháng | |
| **Loại** | Bổ sung (contribution) | |
| **Thời gian đầu tư** | **[Để trống - sẽ tính]** | ⚠️ Biến cần tính |
| **Tỉ suất lợi nhuận** | 12%/năm | Kỳ vọng cao |
| **Lạm phát** | 3.5%/năm | |
| **Mức rủi ro** | Aggressive (Rủi ro cao) | |

### Biến sẽ tính
**Investment Years** - Số năm cần đầu tư để đạt mục tiêu

### Hướng dẫn sử dụng
- Template này phù hợp cho người muốn nghỉ hưu sớm (FIRE)
- FIRE yêu cầu tiết kiệm và đầu tư mạnh mẽ
- Cần chấp nhận rủi ro cao để đạt lợi nhuận cao
- Cân nhắc giảm chi tiêu để tăng tốc độ tích lũy

### Ví dụ kết quả
- **Số năm tính được:** ~18-20 năm (với lợi nhuận 12%/năm)
- **Mục tiêu:** Đạt tự do tài chính trước 50 tuổi

|----------|---------|---------|

| **Loại** | Bổ sung (contribution) | |
| **Thời gian đầu tư** | **[Để trống - sẽ tính]** | ⚠️ Biến cần tính |

**Investment Years** - Số tháng/năm cần tích lũy

## Cách sử dụng Templates

### Bước 1: Chọn Template
- Khi bắt đầu tạo kế hoạch, hệ thống hiển thị danh sách templates
- Chọn template phù hợp với tình huống của bạn
- Hoặc chọn "Bỏ qua, Tự nhập" để nhập thủ công

### Bước 2: Xem lại và Điều chỉnh
- Sau khi chọn template, form sẽ được điền sẵn với giá trị mẫu
- Bạn có thể điều chỉnh bất kỳ giá trị nào cho phù hợp với tình hình thực tế
- Hệ thống sẽ tự động tính lại biến cần thiết

### Bước 3: Tính toán
- Nhấn nút "Tính toán" để hệ thống tính biến còn thiếu
- Xem kết quả và các cảnh báo (nếu có)
- Điều chỉnh lại nếu kết quả không phù hợp

### Bước 4: Tiếp tục Workflow
- Sau khi có kết quả tính toán, tiếp tục các bước tiếp theo:
  - Bước 2: Phân bổ danh mục
  - Bước 3: Tổng quan kế hoạch
  - Bước 4: Theo dõi tiến độ

## Lưu ý quan trọng

1. **Templates chỉ là điểm khởi đầu**: Tất cả giá trị có thể được điều chỉnh
2. **Biến được tính**: Mỗi template chỉ định rõ biến nào sẽ được tính (RRR, Years, Payment, etc.)
3. **Validation**: Hệ thống sẽ cảnh báo nếu kết quả tính toán không thực tế
4. **Customization**: Người dùng có thể lưu template tùy chỉnh của mình
5. **Multiple Plans**: Có thể tạo nhiều plans với các templates khác nhau
